## 실행 순서 권장

세 트랙 병렬 진행 시 **B 트랙(논문)을 가장 먼저 시작** 권장 — 최대 효과:
1. P701 (논문 주제 확정) → 모든 다음 단계의 frame 결정
2. P705 (메트릭 정형화) → P706 비교 실험의 입력
3. P704 (Reproducibility) → 심사위원이 검증 가능
4. P707 → P708 → P709 (논문 finalize)

A 트랙은 하드웨어 도착 일정에 동기화. C 트랙은 발표 부담이 빠지면서 가장 마지막으로 미루는 것이 합리적.
