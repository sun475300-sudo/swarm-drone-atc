Project Summary

문제: 도심 항공 모빌리티(UAM) 시대에 수백 대 드론이 동시 운항할 때 충돌·지연·통제 부재로 인한 공역 안전성 저하.
접근: APF(Artificial Potential Field) 기반 분산 제어 + 중앙 ATC 중재 레이어 + 90초 선제 예측 창을 결합한 하이브리드 공역 통제 시스템.
차별점: 5개 드론 역할(Quadrotor · Heavy-Lifter · Fixed-Wing · Tilt-Rotor · Micro-UAS) 이종 군집 지원 + 8개 시나리오(교차·바람·보티셀·CBS·GNN· 디퓨전·Vertiport) 벤치마크.
구현: Python SimPy(백엔드 검증) + Three.js r175(웹 기반 3D 시뮬레이터, PBR + Bloom + SMAA) + GitHub Pages(영구 배포).
