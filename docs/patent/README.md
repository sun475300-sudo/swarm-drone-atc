## 특허 핵심 구성 (독립항 1 · 청구항 1)

1. **동적 3D 지오펜싱 모듈** — 실시간 이벤트 기반 원통/구체 가상 펜스 생성
2. **APF 경로 계획 모듈** — 목표 인력 + 장애물 반발력 벡터 합성
3. **분산 CD&R 모듈** — 중앙 지시 없이 각 드론이 자율 경로 재계산
4. **WebGPU Compute 병렬 가속 모듈** — APF·CBS·Voronoi 연산의 GPU 병렬화

## 종속 청구항

- 청구항 2: 기상 적응형 파라미터 전환 (강풍 모드)
- 청구항 3: 4계층 아키텍처 (10Hz/1Hz/시뮬/UI)
- 청구항 4: 통신 단절 시 로컬 캐시 기반 회피
- 청구항 5: 펜스 TTL(Time-To-Live) 자동 제거
- 청구항 6: 방법 청구항 (독립항)
- 청구항 7: 강풍 모드 메서드 한정
- 청구항 8: 컴퓨터 판독 가능 매체
